The order_date column is a full date (like 2025-05-14), 
but we want to know which day of the week 
(e.g., Monday, Tuesday) each order was placed.

Extracting day names helps us answer:
➤ "Which days have the most orders?"
➤ "Are weekends busier than weekdays?"

🧠 What this does:
df['order_date'].dt.day_name() converts a full date into a day name string (like "Monday", "Friday").

The new column day_of_week stores this result for each row.

import plotly.graph_objects as go
import seaborn as sns

# Prepare data
pizza_sold_category = df.groupby('pizza_category')['quantity'].sum().sort_values(ascending=False)
colors = sns.color_palette("deep", n_colors=len(pizza_sold_category)).as_hex()

# Create horizontal funnel chart
fig = go.Figure(go.Funnel(
    x=pizza_sold_category.values,  # x now holds values
    y=pizza_sold_category.index,
    orientation='h',               # <- makes it horizontal
    textinfo='value+percent total',
    marker=dict(color=colors)
))

# Set title and size
fig.update_layout(
    title="TOTAL PIZZA SOLD BY PIZZA CATEGORY",
    width=900,
    height=500
)

fig.show()
